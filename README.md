# siqi-webdev-sample

This repository contains a front end coding exercise implemented by **Siqi Guo**.  
It demonstrates skills commonly required for Government of Canada web development work, including semantic HTML, accessibility practices, responsive layout, and content publishing workflows.

---

## Features

- Semantic and accessible HTML5 structure  
- Proper heading hierarchy and ARIA-friendly layout  
- WCAG-oriented markup patterns  
- Responsive layout using the Government of Canada WET BOEW design system  
- Image assets with meaningful alt text  
- Simple, lightweight structure suitable for public-sector content publishing

---

## Tech Stack

- HTML5  
- CSS (Canada.ca / WET BOEW CDN assets)  
- Responsive grid and utility classes  
- No build tools required  

---

## File Structure

```
index.html          Main page markup  
img/                Image assets used in the page  
```

External CSS dependencies are loaded from official WET BOEW and Canada.ca CDNs.

---

## How to Run

1. Clone the repository:

   ```
   git clone https://github.com/<你的github用户名>/siqi-webdev-sample.git
   ```

2. 打开 `index.html` 即可查看，不需要任何安装步骤。

---

## Accessibility Notes

- Logical heading hierarchy (`h1` → `h2` → `h3`)  
- Meaningful alt text for images  
- Layout supports keyboard navigation  
- Markup structure suitable for bilingual content  
- Consistent with typical Government of Canada publishing patterns

---

## About This Sample

This sample represents my approach to structured, maintainable, and accessible front-end development in a public-sector context.  
It focuses on clarity, standards compliance, and reliable content delivery rather than frameworks or heavy tooling.
